# Wiki-Extracter
This program gets noun-list extracted from Wikipedia title list.
We use mecab and NEologd.
##Environment
* OS X EI Capitan: v10.11.4
* Python: v2.7.11
* Anaconda: v2.5.0 (x86_64)
* gcc: v7.3.0
* Cython: v0.24
* mecab
* NEologd
